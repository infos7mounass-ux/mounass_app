# MouNass'App 🚀

Super-app avec 4 modules principaux :
- 💖 Rencontres amoureuses
- ✍️ Vente de textes saisis
- 🛒 Vente de produits & services
- 🏠 Services immobiliers

## ⚙️ Backend
- Django + Django Rest Framework
- Stripe + Flutterwave pour les paiements
- PostgreSQL

## 📱 Frontend
- React Native (Expo)
- Navigation par onglets
- Connexion à l'API Django
